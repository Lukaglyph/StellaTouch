StellaTouch
!

StellaTouch は、直感的な操作でデジタルペイントが楽しめる、モバイル・ブラウザ両対応の多機能ペイントツールです。
レイヤー機能、自由変形による画像取り込み、ネオンエフェクトなど、クリエイティブな作業に必要な機能をブラウザ一つで実現します。

🌟 主な機能
キャンバス設定: 開始時に自由なピクセルサイズを指定可能。

マルチレイヤーシステム:

レイヤーの追加・削除

各レイヤーの不透明度調整

レイヤー名の変更機能

自由変形ツール:

端末内の画像をレイヤーとして取り込み

配置時に「移動・拡大縮小・回転」が可能（透過PNG対応）

多彩な描画ツール:

ブラシ / 発光（ネオン） / 塗りつぶし / スタンプ

リアルタイムなサイズ変更スライダー

高度な編集:

無制限の Undo（戻る） / Redo（進む）

日付・時刻付きの自動ファイル名保存（PNG形式）

🚀 使い方
index.html をブラウザで開きます。

最初にキャンバスの幅と高さを入力し、「開始」ボタンを押します。

画面下のコントロールパネルで色やツールを選び、自由に描画してください。

画像を追加したい場合: 「＋画像レイヤー」ボタンから画像を選択し、青い枠を操作して位置・サイズ・角度を調整。「確定」を押して配置します。

完成したら右上の「保存」ボタンで、日時入りのファイル名で保存されます。

🛠 技術スタック
HTML5 Canvas API: 高速な描画処理

Vanilla JavaScript (ES6+): 外部ライブラリなしで軽量・高速に動作

CSS3: レスポンスデザインとブラー（ぼかし）エフェクトのUI
