StellaTouch
指描き（モバイル）とプロフェッショナルな制作環境（Wacom/ペンタブ）の両方に最適化された、インストール不要のブラウザベースお絵描きアプリケーションです。

🚀 主な機能
📱 モバイル・指描き最適化
手ブレ補正: 指での描画でも滑らかな線を描ける指数平滑法アルゴリズムを搭載。

長押しスポイト: 描画エリアを0.5秒間長押しするだけで、直感的に色を抽出。

左右反転: キャンバスの表示をワンタッチで反転し、デッサン狂いの確認が可能。

ピンチズーム・移動: スマホ特有のジェスチャーによる自由なキャンバス操作。

🎨 高度な制作機能
レイヤーシステム: レイヤーの追加・削除・順序入れ替え・不透明度調整に対応。

描画モード（合成モード）: 「通常」「乗算」「スクリーン」「オーバーレイ」などのブレンドモードをサポート。

筆圧感知: Wacom等のスタイラスペン使用時、筆圧に応じた動的な線の太さの変化に対応。

画像インポート・変形: 外部画像を読み込み、移動・拡大縮小・回転させてキャンバスに統合可能。

特殊ブラシ: 通常のペン、消しゴムに加え、発光ブラシや塗りつぶし（バケツ）ツールを搭載。

⌨️ プロ向けショートカット
制作スピードを落とさないためのキーボード操作に対応しています。

B : ブラシ / E : 消しゴム / G : 塗りつぶし / H : 手のひら

[ / ] : ブラシサイズの変更

Ctrl + Z : 戻る / Ctrl + Y : やり直し

Space : 押し続けている間だけ移動（一時手のひらツール）

🛠️ 技術スタック
Frontend: HTML5 / CSS3 / JavaScript (Vanilla JS)

Graphics: Canvas API (2D Context)

APIs: Pointer Events API (筆圧・マルチタッチ対応)

📦 セットアップ
本プロジェクトは単一のHTMLファイル（または少数の構成ファイル）で動作します。

このリポジトリをクローンまたはダウンロードします。

index.html を任意のモダンブラウザ（Chrome, Safari, Edge等）で開きます。


サーバー設定は不要です。
または、
https://lukaglyph.github.io/StellaTouch/
をを任意のモダンブラウザ（Chrome, Safari, Edge等）で開きます。

📖 使い方
開始: アプリ起動時にキャンバスサイズを入力（または端末サイズを自動取得）して開始します。

描画: 下部のツールバーからペンや消しゴムを選択します。

レイヤー: 右上の「層」ボタンからレイヤーパネルを開き、重ね塗りや不透明度の調整を行います。

保存: 「保存」ボタンを押すと、日付入りのPNGファイルとして書き出されます。
